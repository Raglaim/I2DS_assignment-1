***<span style="color:purple">Question 1:</span>***
Write Python code that takes a number as a string and the base of the number, and prints the decimal value of the number.

***<span style="color:purple">Question 2:</span>***
Write Python code that receives a decimal number and a desired base, and prints the number after converting it to the requested base. The value returned by the code should be of type string.

***<span style="color:purple">Question 3:</span>***
Write Python code that receives two numbers in a given base and the base of those numbers, and prints the sum of the two numbers.

***<span style="color:purple">Question 4:</span>***
Write Python code that receives a binary number and a desired number of digits in the result, and returns the number representing its negative.

## *<span style="color:orange">Functions and their descriptions</span>*

- <span style="color:cyan">strtolist</span> - Turn a string into a list of its characters.
- <span style="color:cyan">listtostr</span> - Combine a list of characters into a single string.
- <span style="color:cyan">swap</span> - Reverse the elements of a list.
- <span style="color:cyan">strtoint</span> - Convert a string of digits into an integer.
- <span style="color:cyan">n_to_10</span> - Convert a number from any base to base 10.
- <span style="color:cyan">n_10_to_base</span> - Convert a base-10 number to any other base.
- <span style="color:cyan">add_in_base</span> - Add two numbers in a given base.
- <span style="color:cyan">return_negative</span> - Calculate the two's complement representation of a binary number.
